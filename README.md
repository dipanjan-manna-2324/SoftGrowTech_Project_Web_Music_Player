#  Web Music Player

A modern **web-based music player** built using **HTML, CSS, and JavaScript**.
It allows users to play music, manage playlists, shuffle tracks, repeat songs, and control volume through an interactive and responsive UI.

This project demonstrates **DOM manipulation, audio API usage, event handling, and UI interaction** in JavaScript.

---

##  Features

* **Play / Pause music**
* **Skip to next or previous track**
* **Shuffle music playback**
* **Repeat current track**
* **Dynamic playlist sidebar**
* **Seek bar for music progress**
* **Volume control & mute option**
* **Real-time running time and duration display**
* **Dynamic background image based on song**
